Learn To Code (Key Notes)

The World Wide Web

The World Wide Web is a collection of html documents. Your browser makes requests via the internet to servers. These requests are using a protocol called HTTP for which the servers respond to the browser displays.

HTML

HTML stands for Hyper Text Markup Language. HTML is made up of text content, markup, references to other documents,and links to other pages. It is the basis for almost every web page.

HTML Tags and Elements

HTML Markup is made up of things called tags. The elements have an opening tag and closing of tags such as "B" for bold and "EM" for italics. Each tag is meant to specify to the computer what you want to do to the text. For example, "B" will make the text bold. Click this to see a video explaining bold tags.

Computers Are Stupid

Programmers say that Computers are stupid not to insult computers but to explain that computers take everything we write literally. Computers need to know exactly what to do because they can't make inferences as humans do. If we don't explain to the computer correctly then this can result in issues between the programmer and the machine.

Different Elements

Inline Vs. Block

Elements are placed into 2 categories; Inline Elements and Block Elements. Inline is a break. An inline element does not start on a new line and only takes up as much space and width as needed. Block is a box. It always starts on a new line and takes up the full width available to the box.

Span Vs. Div

Span and Div are two different types of elements. Span is an inline element and Div is a block element. The "Div" element is used as a container for other HTML elements. It doesn't require an attribute and it doesn't change the look of a HTML element. The "Span" element is used to style a certain text within a larger text element. It also doesn't require an attribute.

HTML Attributes

HTML Attributes

Some elements have additional values called Attributes in their opening tags. These Attributes give additional information to the browser. For example, when you are adding a URL link to your webpage, you would add "href" to the "a" attribute.
